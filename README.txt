# Translator App

An App that takes in a users language and translates it to another. *Additionally can detect languages for bonus points*

## Functionality 

The following **required** functionality is completed:

* [ ] User can write text in the lower region and it is translated in the upper region
* [ ] ViewModel is split but on one app page
* [ ] ML Kit is used to translate the text


The following **extensions** are implemented:

* ML Kit
* Fragment
